# Vibrant Monolith Universal

> An ergonomic, high-contrast theme designed for serious Fullstack & AI Development.
> **Logic-based coloring:** Separating Data (White), Action (Cyan), and Logic (Pink).

![Theme Preview](images/preview.png) *(Note: Anda bisa screenshot tema Anda nanti dan simpan di folder images)*

## ðŸ† Philosophy (The "Why")

Standard themes often confuse **Functions** and **Variables** with similar colors, causing cognitive fatigue during long coding sessions. 

**Vibrant Monolith Universal** solves this using a strict **Architectural Color Logic**:
* **Background:** `#1A1625` (Deep Purple Monolith) - Reduces eye strain while maintaining a vibrant atmosphere.
* **Variables (Data):** `White` - Neutral, readable, distinct from logic.
* **Functions (Action):** `Cyan` - High energy, distinct from data.
* **Keywords (Flow):** `Pink` - Control flow and storage directives.
* **Parameters (Input):** `Orange` - Clearly visible input arguments.

## ðŸ“¦ Installation

1.  Open **Visual Studio Code**.
2.  Go to **Extensions** (Ctrl+Shift+X).
3.  Search for `Vibrant Monolith Universal`.
4.  Click **Install**.
5.  Set the theme: `Ctrl+K` + `Ctrl+T` > Select **Vibrant Monolith Universal**.
